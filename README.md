# LocaBonniProject

# Technologies

- Frontend: HTML, CSS
- Backend: .NET
- Database: PostgreSQL/ Microsoft SQL Developer edition
- Security: JWT (JSON Web Tokens) and more
- Secondary: Docker


# How to run

in the "dotnet" folder terminal write: "dotnet run"
and open the localhost written on the terminal


# Notes

The frontend is in the "wwwroot" file as it is the 
default root name for .NET to search for frontend
