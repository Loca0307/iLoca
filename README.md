# LocaBonniProject

# Technologies

- Frontend: HTML, CSS
- Backend: asp.NET
- Database: PostgreSQL/ Microsoft SQL Developer edition
- Security: JWT (JSON Web Tokens) and more
- Testing: xUnit, NUnit
- Secondary: Docker

# Notes

The frontend is in the "wwwroot" file as it is the 
default root name for asp.NET to search for frontend

# asp.NET

in the "dotnet" folder terminal write: "dotnet run"
and open the localhost written on the terminal


# Docker

To build the docker image:
- docker build -t "image name"

To run the image and create a container:
- docker run -p "host port":"container port" "image name"
